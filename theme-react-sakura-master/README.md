## 快速开始

### 克隆项目到本地

```language
clone https://github.com/Lhanman/sakura.git
```
### 打开 helloblog-sakura 代码后进行构建

使用 yarn

### 项目初始化
```language
yarn install 
```

### 启动项目

```language
yarn run start
```
## 本地调试设置代理

在 package.json 中 proxy 字段 只需设置域名和端口就行<br/>

在 /src/lib/axios.js 中 设置请求前缀

## 打包项目

```language
yarn run build
```
